Place the Qualcomm-exported LiteHRNet TFLite file in this directory.

The file must be named exactly:
  litehrnet_pose_256x192.tflite

Expected model contract:
  input  : [1, 256, 192, 3]
  output : [1, 17, 3] or [1, 1, 17, 3]

Without this file the app runs in demo mode (MockPoseEstimator) with
an animated skeleton.

To verify NPU is active after adding the model, filter Logcat for:
  Tag: FitForm/LiteRT
  Look for: "NNAPI delegate created" and LiteHRNet per-frame latency logs
